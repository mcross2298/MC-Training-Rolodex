📋 Program Guide

Everything Under the Kitchen Sink

6 distinct training splits · station-anchored for commercial gym efficiency

Every Session, Same Blueprint
Every workout in all 6 splits runs the same 10-exercise blueprint — the position on the list tells you the intensifier, no matter which split or muscle group you're training.
1–2
Compound (standalone) — heavy, no superset, no station-sharing
3–5
Tri-Set — 3 exercises back to back at one station
6–7
Superset — 2 exercises back to back at one station
8
Cluster Set — base sets, then short-rest micro-sets to extend the last set
9
Drop Set — base sets, then strip weight and continue immediately
10
Finisher — bodyweight, no station constraint
Station-Anchoring
Every tri-set and superset stays inside a single, minimal footprint — one piece of equipment or one square area. No walking the floor mid-block, no hogging two stations at once.
A · DB / Bench
All-dumbbell work at one adjustable bench
B · DB + Fixed Bar
EZ-bar or fixed barbell paired with DB/bodyweight, same bench
C · Single Machine
One machine + bodyweight or DBs brought to it
D · Single Cable Column
One stack, swap attachments for each move
